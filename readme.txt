Future post, adjust the modify date to post date.

== Description ==

When this plug-in is enabled, when future post, adjust the modify date to post date.

== Installation ==

You can install this plugin directly from your WordPress dashboard:

1. Go to the **Plugins** menu and click **Add New**.
1. Search for **Ys-Fp_md2pd**.
1. Click **Install Now** next to the **Ys-Fp_md2pd** plugin.
1. Activate the plugin.
